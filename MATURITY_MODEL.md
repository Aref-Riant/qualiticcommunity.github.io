# Qualitic Maturity Model

Qualitic Maturity model is consists of three parts.

+ Baseline CI Pipeline
+ Maturity Items
+ Maturity Levels

## Baseline CI Pipeline

This pipeline consists of five steps.

+ Commit Stage

+ Code Review

+ Acceptance Test Stage

+ Performance Test Stage

+ Staging
